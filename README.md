# NeoVim Config

## Install
Config
Clone this to `~/.config` directory as `nvim`.
`git clone git@github.com:kamkejj/nvim_config.git nvim`

## Shortcuts
<leader> = " " -> Space bar
C = control

Go to previous
`<leader>pv`

File Search
`<leader>ff` -> opens search window

Git 
status: `:Git`

Harpoon 
open: `C-e` -> opens Harpoon window
`<leader>a` -> adds file to Harpoon
